方法增强

用法

在切面类中加入对应时机的处理方法, 可以实现目标方法的增强.

支持的注解

注解 说明
@Before 方法执行前调用
@After 方法执行后调用
@AfterReturning 方法返回后调用
@AfterThrowing 方法抛出异常后调用
@Around 方法环绕, 全权接管目标方法的调用过程

异常处理

想通过AOP实现全局异常处理, 使用@Around并且将调用过程用try-catch包裹即可.
